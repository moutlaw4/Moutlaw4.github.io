Analyzing Web Design with CRAP Principles

Website Design Evaluation

Mobbin

Mobbin is a curated design library that showcases real-world examples of mobile and web apps. It helps designers explore professional UI/UX patterns and trends across a variety of popular applications.

  • Contrast: Excellent use of whitespace, typography, and color contrast to highlight key elements.
  • Repetition: Consistent font styles, color palette, and spacing create a cohesive visual system.
  • Alignment: Perfectly aligned content grids that make navigation intuitive and aesthetically pleasing.
  • Proximity: Related elements (like search, categories, and results) are grouped clearly, improving usability.
